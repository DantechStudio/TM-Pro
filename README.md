## how to change it to ```.exe``` file
- open CMD
- run :
```diff
+  py -m pip install pyinstaller
```
- close CMD
- go to project folder
- then do ```Shift + right click```
- then click on ```open PowerShell window here```
- run :
```diff
+  py -m PyInstaller -F -w -i icon.ico main.py
```
- then create shortcut of ```dist/main.exe``` on desktop

## Attention
- if you want to make ```.exe``` file, you need to have the IDle installed ```Python```
- don't delete ```Tasks.db``` file
